A 30-day SQL learning journey using **Google BigQuery** and the **TheLook eCommerce** public dataset. Each day covers one SQL topic applied to a realistic business context.

---

## Dataset

**`bigquery-public-data.thelook_ecommerce`**

A simulated e-commerce platform dataset available for free via BigQuery Public Datasets.

| Table | Description |
|---|---|
| `users` | Customer data (name, location, age, gender) |
| `orders` | Order transactions (status, date, user) |
| `order_items` | Item-level order details (product, price, status) |
| `products` | Product catalog (name, category, brand, price) |
| `inventory_items` | Warehouse stock data |
| `events` | User behavior tracking (page views, add to cart, etc.) |
| `distribution_centers` | Distribution center locations |
