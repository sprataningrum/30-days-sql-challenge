Repositori ini mendokumentasikan perjalanan belajar SQL selama 30 hari menggunakan **Google BigQuery** dan dataset publik **TheLook eCommerce**. Setiap hari mencakup satu topik SQL dengan konteks bisnis yang relevan.

---

## Dataset

**`bigquery-public-data.thelook_ecommerce`**

Dataset simulasi platform e-commerce yang tersedia gratis di BigQuery Public Datasets.

| Tabel | Deskripsi |
|---|---|
| `users` | Data pelanggan (nama, lokasi, usia, gender) |
| `orders` | Data transaksi order (status, tanggal, user) |
| `order_items` | Detail item per order (produk, harga, status) |
| `products` | Katalog produk (nama, kategori, brand, harga) |
| `inventory_items` | Stok produk di gudang |
| `events` | Event tracking (page view, add to cart, dll) |
| `distribution_centers` | Lokasi pusat distribusi |
